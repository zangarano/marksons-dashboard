# Markson's Dashboard

Internal sales analytics dashboard for Greece location.

## Reports

- **Attachment Rates** — Delivery and protection attachment by salesperson, with profit % and average sale. Upload Storis and Trax exports and filter by date range.

## Adding to GitHub Pages

1. Create a new repo on GitHub (e.g. `marksons-dashboard`)
2. Upload all files maintaining the folder structure:
   ```
   index.html
   css/global.css
   reports/index.html
   reports/attachment-rates.html
   ```
3. Go to Settings → Pages → set source to `main` branch, `/ (root)`
4. Dashboard will be live at `https://zangarano.github.io/marksons-dashboard/`

## How to use

1. Open the dashboard and go to **Reports → Attachment Rates**
2. Upload your Storis export (Report Written Sales Dollars) and Trax export (Salespeople)
3. Use the date filter buttons or custom date range to slice the data
4. Print / export button generates a PDF — charts appear on page 2
